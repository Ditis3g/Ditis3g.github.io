조사 ① 기술 흐름 — LLM에서 피지컬 AI까지

최신 모델을 외우지 말고 '계보'를 읽어라 — 각 단계의 남은 한계가 다음 단계를 낳는다
피지컬 AI는 아직 하나의 정답으로 수렴하지 않은 분야 — 오늘의 SOTA가 몇 달 뒤 대체된다. 그래서 새 기술은 세 가지 질문으로 읽는다: ① 어떤 문제를 풀려 등장했나 → ② 어떤 아이디어로 풀었나 → ③ 무엇을 여전히 못 푸나. ③의 답(남은 한계)이 곧 다음 기술이 태어난 이유다.
단계 정체 (입력→출력) ＋ 더한 능력 △ 남은 한계 → 다음 촉발
LLM 텍스트 → 텍스트 (출발점) 명령을 작업 순서로 계획 장면을 직접 못 봄 → 시각 필요
VLM 텍스트+이미지 → 텍스트 ＋시각 (이미지 이해) 이해만 할 뿐 행동을 못 냄 → 행동 출력 필요
VLA 시각+언어 → 행동 ＋행동 출력 (관절/그리퍼 직접) — RT-2·OpenVLA·π0 정답이 여럿·오차 누적 → 행동 표현 기법 필요
↳ VLA 내부 기법 행동을 어떻게 낼까 액션 청크 · 디퓨전/플로우 매칭 · 언어 중간표현 (RT-H·ECoT) 장기 계획·재계획 약함 → 상위 층 필요
계층형 시스템 상위(추론) + 하위(제어) ＋상위 추론층 — 느린 판단 + 빠른 제어 (Helix·Gemini Robotics) 시뮬↔현실 격차·데이터 부족 → 경험 확보 필요
피지컬 AI 시스템 두뇌 + 몸·경험의 순환 ＋데이터·시뮬레이션·배포·지속학습 진행형 — 아직 완성되지 않음
· 아래 단계는 위 단계를 대체하지 않고 품고 간다 (대체 ✕, 누적 ○) — 계층형 시스템에서도 VLA는 '손발'로 계속 쓰인다
· 기회는 모델 자체보다 데이터 평가·시뮬 환경·운영·지속학습 같은 시스템의 빈틈에 더 많이 남아 있다
진화의 동력은 "더 큰 모델"이 아니라 앞 방식의 약점을 메우는 방향이었다 — 새 논문·모델은 이름이 아니라 계보 위의 위치로 판단한다.

조사 ② 로봇은 어떻게 '행동'을 배우는가

VLA 학습 3대 패러다임 · 영상→행동 4가지 전환 기술
학습 3대 패러다임
"비싼 로봇 데이터를 얼마나 덜 쓰느냐"가 방식을 가른다
사람이 로봇을 원격조종해 모으는 데이터는 시간당 $50~100. 이 비용 구조가 아래 세 갈래의 학습 전략을 만들었다.
1

직접 정책 학습 — 지금 가장 실용적 — 로봇이 모은 (관찰, 행동) 쌍을 그대로 학습. RT-2·OpenVLA·π0. 성능은 높지만 비싼 로봇 데이터를 가장 많이 소비.

2

영상 사전학습 — 차세대 R&D 핵심 — 행동 라벨 없는 인터넷 영상으로 먼저 '감'을 잡고, 소량의 로봇 데이터로 마무리. LAPA는 행동 라벨을 약 30배 절감, GR-2는 3,800만 영상 학습으로 97% 성공률.

3

월드모델 — 가장 야심찬 미래형 — "행동하면 세상이 어떻게 변하나"를 모델이 직접 상상·시뮬레이션. Dreamer·Genie 2·NVIDIA Cosmos. 잠재력은 크지만 비용도 가장 크다.

→ "CCTV 영상으로 그냥 가르치면 안 되나?" — 행동·모달리티·도메인·인과성 4가지 갭 때문에 그대로는 불가. 이 갭을 줄이는 것이 오른쪽의 전환 기술이다.
영상→행동, 4가지 전환 기술
행동 라벨 없는 영상에서 행동을 뽑아내는 방법
영상엔 행동 라벨·물리 신호(힘/관절 상태)가 없고, 사람 손은 로봇 그리퍼와 다르다(embodiment gap). 이 결핍을 메우는 방식이 네 갈래로 경쟁 중.
1

잠재 행동 학습 (LAPA) — 프레임 간 변화를 가짜 행동 라벨로 인코딩. 무라벨 사전학습이 라벨로 학습한 OpenVLA를 능가(50.1% vs 43.9%).

2

월드모델 사전학습 (V-JEPA 2 · Cosmos) — 미래 상태 예측으로 동역학을 학습. V-JEPA 2는 100만 시간+ 영상을 라벨 없이 학습, 처음 보는 로봇에서 zero-shot 동작. Cosmos는 인과적 영상 토큰화(과거만 보고 미래 예측)로 Physical AI 파운데이션 지향.

3

2D 단서 추출 — 포인트 트랙·마스크·옵티컬 플로우를 중간 표현으로 사용해 행동 학습을 유도.

4

3D 복원 + 리타게팅 (H2R · X-Humanoid) — 사람 손 자세·궤적을 3D 복원해 로봇으로 변환. 사람 영상을 휴머노이드 영상으로 바꿔 360만 프레임 학습 데이터 생성.

→ 산업 표준은 "사전학습은 풍부한 영상, 파인튜닝은 소량 로봇 데이터"로 수렴 중 (NVIDIA GR00T·Cosmos, π 시리즈).
실무 적용 로드맵: A) 시뮬 사전학습 + OpenVLA 파인튜닝으로 1~2주 PoC → B) 영상 잠재행동 사전학습을 얹어 라벨링 비용 30~40% 절감 → C) 월드모델 결합은 R&D 동향 관찰. 경쟁력은 모델 구조가 아니라 데이터 전환 전략이 가른다.

조사 ③ 왜 휴머노이드가 주목받는가

형태의 문제가 아니라 데이터와 경제 논리의 문제
1

세상이 사람 몸 기준으로 설계돼 있다

공장 라인·계단·문손잡이·공구가 전부 사람 기준. 사람 형태면 인프라 개조 비용 없이 기존 작업 환경에 그대로 투입 가능 — "전용 로봇 N종 개발"이 "범용 1종 + 소프트웨어" 문제로 바뀜.

2

영상 학습에 가장 유리한 몸 (조사 ②와 직결)

인터넷에 무한한 것은 "사람이 일하는 영상". 영상 학습의 최대 장벽인 신체 구조 차이(embodiment gap)가 사람을 닮은 휴머노이드에서 가장 작다. 조사 ②의 전환 기술이 가장 잘 먹히는 폼팩터.

3

범용 모델의 스케일 법칙과 맞물림

VLA는 데이터를 먹일수록 좋아지는 대형 모델. 폼팩터가 통일된 범용 몸이 있어야 데이터가 한곳에 쌓이는 데이터 플라이휠(배치→데이터→모델 개선)이 돈다. Tesla·Figure·1X가 자사 공장을 플라이휠로 운영 중.

4

하드웨어 단가 붕괴 + 자본 유입

EV 공급망 덕에 휴머노이드 BOM이 $20만(2020) → $3만 이하(2026), Unitree G1은 $1.6만. 2024~25년 휴머노이드 스타트업에 $70억+ 투자 유입 (OpenAI·NVIDIA·Samsung 직접 출자).

휴머노이드는 가장 어려운 폼팩터지만, 사람의 인프라 · 사람의 영상 데이터 · 대형 모델의 스케일 법칙을 전부 그대로 물려받을 수 있는 유일한 형태다 — 영상 사전학습이 표준이 되는 시대일수록 이 이점은 커진다.
